# wpy-annual-review

微北洋年度总结（2025）前端页面。

## 启动项目

```bash
git clone https://git.twt.edu.cn/FANCC/wpy-annual-review.git
pnpm install
pnpm run dev
```

## 打包项目

```bash
pnpm run build
```
